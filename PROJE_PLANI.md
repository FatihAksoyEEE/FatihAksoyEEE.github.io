# Fatih Aksoy - Kişisel Portföy Web Sitesi
## Elektrik-Elektronik Mühendisliği Portfolyosu

## 📋 Proje Özeti
Elektrik-Elektronik Mühendisliği öğrencisi Fatih Aksoy için **profesyonel, teknik odaklı ve interaktif** bir kişisel portföy web sitesi. Bu site sadece bir CV değil, mühendislik yetkinliğini kanıtlayan **canlı bir showcase**'tir.

---

## 🎯 Proje Hedefleri
- **Teknik Yetkinliği Kanıtlamak**: "Ne bildiğin" değil, "ne yaptığın" önemlidir
- **Profesyonel Online Varlık**: İşverenler ve teknik yöneticiler için optimize edilmiş
- **Bilgi Paylaşımı**: Blog ve teknik notlarla öğrenme sürecini belgelemek
- **İnteraktif Deneyim**: Canlı veri, simülasyonlar ve dinamik içerik
- **Sosyal Kanıt**: Sertifikalar, GitHub aktivitesi, LinkedIn entegrasyonu

#### Proje Detay Sayfası/Modal:
Her proje için **kapsamlı teknik dokümantasyon**:

1. **Proje Özeti**
   - Problem tanımı
   - Çözüm yaklaşımı
   - Hedefler

2. **Teknik Detaylar**
   - **Donanım**: Hangi mikrodenetleyici? (Arduino Uno, STM32F4, ESP32 vb.)
   - **Haberleşme**: Hangi protokoller? (I2C, SPI, UART, WiFi, Bluetooth)

   - **Güç Kaynağı**: Voltaj seviyeleri, regülatörler

3. **Yazılım Mimarisi**
   - Kullanılan kütüphaneler
   - Algoritma açıklaması
   - **Kod Blokları**: Kritik fonksiyonlar (syntax highlighting ile)
   - **GitHub Linki**: Tam kaynak kod

4. **Görsel Materyaller**
   - **Devre Şemaları**: Fritzing veya profesyonel şema
   - **PCB Tasarımı**: 3D render veya gerçek PCB fotoğrafı
   - **Prototip Fotoğrafları**: Çalışan sistem
   - **Demo Video**: Kısa (30-60 saniye) çalışma videosu (YouTube embed veya lokal)

5. **Karşılaşılan Sorunlar ve Çözümler** ⭐
   - **Problem**: Hangi hata/zorluk yaşandı?
   - **Çözüm**: Nasıl çözüldü?

## 🎨 Tasarım Konsepti

### Renk Paleti - "Elektrik Mavisi Teması"
- **Dark Mode (Varsayılan)**:
  - Arka plan: `#0a0e27` (derin lacivert)
  - Kartlar/Paneller: `#1a1f3a` (koyu mavi-gri)
  - Vurgu rengi: `#00d4ff` (elektrik mavisi - neon efekti)
  - Alternatif vurgu: `#ff6b35` (turuncu - enerji)
  - Metin: `#ffffff` (beyaz), `#b8c1ec` (açık gri-mavi)
  - Başarı: `#00ff88` (yeşil)
  - Uyarı: `#ffd700` (altın sarısı)
  
- **Light Mode (Opsiyonel)**:
  - Arka plan: `#f8f9fa`
  - Kartlar: `#ffffff`
  - Vurgu: `#0066cc` (koyu mavi)
  - Metin: `#1a1f3a`

### Gradient Kullanımı
- **Hero background**: `linear-gradient(135deg, #0a0e27 0%, #1a1f3a 50%, #2a3f5a 100%)`
- **Butonlar**: `linear-gradient(90deg, #00d4ff 0%, #0099cc 100%)`
- **Kartlar**: Hafif gradient + glassmorphism

### Tipografi
- **Başlıklar**: 
  - Primary: **Inter** (modern, profesyonel)
  - Alternative: **Outfit** (geometrik, temiz)
- **Body Text**: **Roboto** (okunabilir)
- **Kod Blokları**: **Fira Code** (ligatures ile)
- **Teknik Terimler**: **JetBrains Mono**

### Görsel Stil - "Avant-Garde Engineering"
- **Glassmorphism**: 
  - `background: rgba(26, 31, 58, 0.7)`
  - `backdrop-filter: blur(10px)`
  - `border: 1px solid rgba(255, 255, 255, 0.1)`
  
- **Micro-animations**:
  - Hover: Scale(1.05), glow efekti
  - Scroll: Fade-in, slide-up
  - Loading: Skeleton screens
  
- **Smooth Transitions**: `transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1)`

- **Responsive Design**: 
  - Mobile-first yaklaşım
  - Breakpoints: 640px, 768px, 1024px, 1280px

- **Dark Mode Toggle**: 
  - Animated switch
  - Smooth color transitions
  - LocalStorage persistence

### İkon Sistemi
- **Font Awesome 6** veya **Lucide Icons**
- **Devicon**: Teknoloji logoları (Python, C++, Arduino vb.)
- **Simple Icons**: Sosyal medya (LinkedIn, GitHub)
